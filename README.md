# MultiRemote ESP32
## Matériel
- Adaptateur USB To Uart 5V/3V3
- Câble USB-A vers USB-micro

## Installation
- Positionner le micro-switch de Adaptateur USB To Uart 5V/3V3 sur 3V3.
- Connecter le câble USB-A vers USB-micro au PC et sur l'adaptateur.
